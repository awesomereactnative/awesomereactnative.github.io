Displaying Image from URL in React Native

In this blog post I will show you how to display an image from URL in React Native application.

First step is create a new React Native project by command below on your terminal.

npx react-native init DisplayImageFromURL

Open created project on your code editor and clear current code in App.js file then implement App.js as steps below.

Import components 

import React from 'react'
import { View, Image } from 'react-native'

Display image from https://awesomereactnative.com/images/react-logo.png by using Image component. 

class App extends React.Component {
  render() {
    return(
      <View>
        <Image source={{ uri: 'https://awesomereactnative.com/images/react-logo.png' }}
          style={{ width: 400, height: 400 }} />
      </View>
    )
  }
}

Export the App component 

export default App

The complete App.js file as below 

import React from 'react'
import { View, Image } from 'react-native'

class App extends React.Component {
  render() {
    return(
      <View>
        <Image source={{ uri: 'https://awesomereactnative.com/images/react-logo.png' }}
          style={{ width: 400, height: 400 }} />
      </View>
    )
  }
}

export default App

Run application in your simulator by command

npx react-native run-android
